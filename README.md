# Oh, hai Maulgar!

A World of Warcraft (2.4.3) addon for managing assignments
for **High King Maulgar** encounter in **Gruul's Lair** raid.


## Usage

Type: `/hkm`

**NOTE**: You must be in a raid group to be able to assign players.
